# reservoir_data

`reservoir_data` is an object-oriented Python library for discovering, loading, representing, querying, and selectively exporting reservoir simulation data.

The current implementation is at milestone M1: project foundation and case discovery. It can detect and catalog known case files without parsing heavy simulator payloads. Format readers are added only when real behavior and validation are implemented.
